## CRM & Customer Management

- **[frappe/erpnext](https://github.com/frappe/erpnext)** — Full ERP suite, heavily used by Indian SMEs. Open source, self-hostable.
- **[supabase/supabase](https://github.com/supabase/supabase)** — Postgres-backed backend used by many Indian SME SaaS builders.

## Invoice & GST Compliance

- **[signalx/gst-in](https://github.com/signalx/gst-in)** — GSTIN validation and lookup utilities for Python.
- **[nikhilkumarsingh/gstin-validator](https://github.com/nikhilkumarsingh/gstin-validator)** — Lightweight Python GSTIN validator with checksum verification.
- **[reportlab/reportlab](https://github.com/MrBitBucket/reportlab-mirror)** — PDF engine used for GST invoice generation in Python.

## WhatsApp & Communication Parsing

- **[sme-inbox-parser](https://github.com/challanamuskan/sme-inbox-parser)** ⭐ — Parse WhatsApp, Gmail, and website leads into CRM CSV. Built for Indian SME order management.
- **[adithya-s-sekhar/advanced-whatsapp-chat-analyzer](https://github.com/adithya-s-sekhar/advanced-whatsapp-chat-analyzer)** — WhatsApp chat stats and analytics in Python.
- **[twilio/twilio-python](https://github.com/twilio/twilio-python)** — WhatsApp Business API via Twilio, widely used for SME order bots.

## Payments & Fintech

- **[razorpay/razorpay-python](https://github.com/razorpay/razorpay-python)** — Official Razorpay SDK. Most common payment gateway for Indian SMEs.
- **[cashfree/cashfree-sdk-python](https://github.com/cashfree/cashfree-sdk-python)** — Cashfree Payments Python SDK.
- **[paytm/paytm-pg-python-sdk](https://github.com/paytm/paytm-pg-python-sdk)** — Paytm Payment Gateway SDK.

## Inventory & Supply Chain

- **[frappe/erpnext](https://github.com/frappe/erpnext)** — Stock ledger, warehouse management, purchase orders.
- **[inventree/InvenTree](https://github.com/inventree/InvenTree)** — Lightweight open source inventory management with REST API.

## Analytics & Reporting

- **[metabase/metabase](https://github.com/metabase/metabase)** — Self-hosted BI tool, popular for SME dashboards on top of Supabase/MySQL.
- **[streamlit/streamlit](https://github.com/streamlit/streamlit)** — Python-first dashboarding, widely used for Indian SME internal tools.

## E-Commerce & Catalogues

- **[saleor/saleor](https://github.com/saleor/saleor)** — Headless e-commerce platform, used by Indian D2C brands.
- **[medusajs/medusa](https://github.com/medusajs/medusa)** — Shopify alternative, increasingly adopted by Indian SME online stores.

## Government & Regulatory APIs

- **[GSTN API](https://developer.gst.gov.in/)** — Official GST Network API for return filing, e-Way bills, e-invoicing.
- **[India Stack](https://indiastack.org/)** — Aadhaar, UPI, DigiLocker API documentation.
- **[NIC e-Invoice API](https://einvoice1.gst.gov.in/Others/APIList)** — e-Invoice generation API (mandatory for turnover > ₹5Cr).
- **[Open Government Data Platform](https://data.gov.in/)** — Public datasets useful for market research tools.

## Developer Infrastructure

- **[supabase/supabase](https://github.com/supabase/supabase)** — Best backend stack for Indian SME SaaS builders.
- **[pocketbase/pocketbase](https://github.com/pocketbase/pocketbase)** — Single-file backend, good for small offline-first SME apps.
- **[nocodb/nocodb](https://github.com/nocodb/nocodb)** — Airtable alternative, often used for SME internal tools without code.

## Learning Resources

- **[IndiaStack Developer Docs](https://indiastack.org/)** — UPI, BBPS, Aadhaar APIs explained.
- **[GST Developer Portal](https://developer.gst.gov.in/)** — Official API docs for all GST integrations.
- **[Razorpay Dev Docs](https://razorpay.com/docs/)** — Best payment integration guide for Indian developers.
