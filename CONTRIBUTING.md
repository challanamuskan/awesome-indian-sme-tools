# Contributing to Awesome Indian SME Tools

Thanks for helping grow this list!

## How to add a tool

1. Fork this repo
2. Edit `README.md`
3. Add your tool under the correct section in this format:
   ```
   **[Tool Name](https://github.com/owner/repo)** — One sentence description. No marketing language.
   ```
4. Open a PR with title: `Add: tool-name`

## Rules

- ✅ Must be open source OR have a genuinely useful free tier
- ✅ Must be relevant to Indian SME context (GST, INR, WhatsApp, UPI, Aadhaar, MSME)
- ✅ Must be actively maintained (last commit < 2 years ago)
- ❌ No paid-only tools
- ❌ No self-promotion without a real project behind it
- ❌ No duplicate entries

## Sections you can add to

If your tool doesn't fit any existing section, propose a new one in your PR description.
