## Our Own Tools

Built and maintained by [Muskan Challana](https://muskanchallana.vercel.app) — freelance AI/CRM builder for Indian SMEs (DPIIT-registered venture):

- **[indian-sme-crm-template](https://github.com/challanamuskan/indian-sme-crm-template)** ⭐ — Full business OS for Indian SMEs: CRM, GST invoices, stock, WhatsApp orders, compliance calendar, marketing, supply chain, finance P&L, global export tools. [Live Demo →](https://crm-template.streamlit.app)

## CRM & Customer Management

- **[frappe/erpnext](https://github.com/frappe/erpnext)** — Full ERP suite, heavily used by Indian SMEs. Open source, self-hostable, includes CRM, HR, inventory.
- **[supabase/supabase](https://github.com/supabase/supabase)** — Postgres-backed backend used by many Indian SME SaaS builders as their CRM data layer.
- **[twenty/twenty](https://github.com/twentyhq/twenty)** — Open-source Salesforce alternative. Clean UI, self-hostable, REST API-friendly.
- **[Corteza](https://github.com/cortezaproject/corteza)** — Low-code CRM platform with Indian SME deployments. Handles multi-lingual data.

## Invoice & GST Compliance

- **[signalx/gst-in](https://github.com/signalx/gst-in)** — GSTIN validation and lookup utilities for Python.
- **[nikhilkumarsingh/gstin-validator](https://github.com/nikhilkumarsingh/gstin-validator)** — Lightweight Python GSTIN validator with checksum verification.
- **[reportlab/reportlab](https://github.com/MrBitBucket/reportlab-mirror)** — PDF engine for GST invoice generation in Python.
- **[invoice-ninja/invoiceninja](https://github.com/invoiceninja/invoiceninja)** — Self-hostable invoice + billing platform with GST support.
- **[akaunting/akaunting](https://github.com/akaunting/akaunting)** — Open source accounting with invoice, tax, and multi-currency for small businesses.

## Marketing & Growth

- **[mautic/mautic](https://github.com/mautic/mautic)** — Open source marketing automation. Email campaigns, lead scoring, drip sequences.
- **[listmonk/listmonk](https://github.com/knadh/listmonk)** — High-performance self-hosted email newsletter and mailing list manager. Used by Indian SMEs for bulk mailers.
- **[plausible/analytics](https://github.com/plausible/analytics)** — Privacy-first website analytics. Simple embed in Streamlit or any web page.
- **[umami-software/umami](https://github.com/umami-software/umami)** — Open source, self-hostable Google Analytics alternative.
- **[posthog/posthog](https://github.com/PostHog/posthog)** — Product analytics + session recording. Good for SME e-commerce and SaaS.
- **[Matomo](https://github.com/matomo-org/matomo)** — Self-hostable analytics with India-specific campaign tracking.

## WhatsApp & Communication

- **[sme-inbox-parser](https://github.com/challanamuskan/sme-inbox-parser)** ⭐ — Parse WhatsApp, Gmail, and website leads into CRM CSV. Built for Indian SME order management.
- **[adithya-s-sekhar/advanced-whatsapp-chat-analyzer](https://github.com/adithya-s-sekhar/advanced-whatsapp-chat-analyzer)** — WhatsApp chat stats and analytics in Python.
- **[twilio/twilio-python](https://github.com/twilio/twilio-python)** — WhatsApp Business API via Twilio. Widely used for SME order bots.
- **[Baileys](https://github.com/WhiskeySockets/Baileys)** — Lightweight WhatsApp Web API for Node.js. Used for order automation bots.
- **[wabot-shop](https://github.com/Streen9/wabot-shop)** — WhatsApp-based shop bot for small businesses.

## Finance & Accounting

- **[akaunting/akaunting](https://github.com/akaunting/akaunting)** — Full open-source accounting: invoices, expenses, payroll, bank reconciliation. INR-ready.
- **[dolibarr/dolibarr](https://github.com/Dolibarr/dolibarr)** — ERP + CRM + accounting + HR for SMEs. PHP-based, widely deployed in India.
- **[frappe/books](https://github.com/frappe/books)** — Desktop double-entry accounting for small businesses. Made in India, GST-ready.
- **[invoice-ninja](https://github.com/invoiceninja/invoiceninja)** — Invoicing, payments, time tracking for service businesses.
- **[firefly-iii/firefly-iii](https://github.com/firefly-iii/firefly-iii)** — Self-hosted personal + business finance manager. Good for HNI portfolio tracking.
- **[ghostfolio/ghostfolio](https://github.com/ghostfolio/ghostfolio)** — Open source wealth management for HNIs. Tracks stocks, MF, ETF with INR support.

## Supply Chain & Inventory

- **[frappe/erpnext](https://github.com/frappe/erpnext)** — Stock ledger, warehouse management, purchase orders. Used by Indian manufacturers.
- **[inventree/InvenTree](https://github.com/inventree/InvenTree)** — Lightweight open source inventory management with REST API. Easy to embed in existing tools.
- **[odoo/odoo](https://github.com/odoo/odoo)** — Full supply chain: procurement, warehouse, manufacturing, delivery. Community edition is free.
- **[StockPy](https://github.com/nickmvincent/stockpy)** — Python toolkit for inventory signal processing and demand forecasting.
- **[openboxes/openboxes](https://github.com/openboxes/openboxes)** — Supply chain management for SMEs with warehouse + purchase + shipping modules.

## Content & Communication Tools

- **[Typebot](https://github.com/baptisteArno/typebot.io)** — Open source chatbot builder. Deploy as WhatsApp/web lead capture for SMEs.
- **[n8n](https://github.com/n8n-io/n8n)** — Workflow automation connecting WhatsApp, Gmail, Sheets, CRM. Self-hostable.
- **[Cal.com](https://github.com/calcom/cal.com)** — Open source scheduling. Useful for service businesses and consultants.

- **[GrapesJS](https://github.com/GrapesJS/grapesjs)** — Open source web page / email builder for building SME content pages.

## Global Trade & Export

- **[openexchangerates](https://github.com/metadevpro/openexchangerates)** — Currency exchange rate library. Integrates with INR for export invoicing.
- **[forex-python](https://github.com/MicroPyramid/forex-python)** — Python library for INR↔USD/EUR/AED/GBP real-time and historical rates.
- **[DGFT Developer APIs](https://www.dgft.gov.in/)** — Directorate General of Foreign Trade API for IEC, RCMC, trade notices.
- **[India EXIM Portal](https://www.exim.gov.in/)** — Export/import policy lookup, duty calculator resources.
- **[shiprocket-python](https://github.com/ashishb/shiprocket-python)** — Python SDK for Shiprocket — India's most-used e-commerce shipping platform.
- **[delhivery-sdk](https://github.com/Delhivery/delhivery-python-sdk)** — Python SDK for Delhivery courier API. Track, ship, return.

## Payments & Fintech

- **[razorpay/razorpay-python](https://github.com/razorpay/razorpay-python)** — Official Razorpay SDK. Most common payment gateway for Indian SMEs.
- **[cashfree/cashfree-sdk-python](https://github.com/cashfree/cashfree-sdk-python)** — Cashfree Payments Python SDK with UPI, netbanking, wallet support.
- **[paytm/paytm-pg-python-sdk](https://github.com/paytm/paytm-pg-python-sdk)** — Paytm Payment Gateway SDK.
- **[UPI Deep Link Generator](https://developer.npci.org.in/)** — NPCI's UPI specification for building UPI payment links and QR codes.
- **[phonepe-sdk-python](https://github.com/PhonePe/PhonePe-pg-sdk-python)** — PhonePe payment gateway SDK.

## Analytics & Business Intelligence

- **[metabase/metabase](https://github.com/metabase/metabase)** — Self-hosted BI tool. Connects to Supabase/MySQL for SME dashboards.
- **[streamlit/streamlit](https://github.com/streamlit/streamlit)** — Python-first dashboarding. Most popular for Indian SME internal tools.
- **[apache/superset](https://github.com/apache/superset)** — Enterprise-grade BI for SMEs that have outgrown spreadsheets.
- **[redash/redash](https://github.com/getredash/redash)** — Self-hosted dashboards from SQL. Good for Supabase + SME analytics.
- **[grafana/grafana](https://github.com/grafana/grafana)** — Real-time dashboards. Used for operational monitoring in larger SMEs.

## Government & Regulatory APIs

- **[GSTN API](https://developer.gst.gov.in/)** — Official GST Network API for return filing, e-Way bills, e-invoicing.
- **[India Stack](https://indiastack.org/)** — Aadhaar, UPI, DigiLocker API documentation.
- **[NIC e-Invoice API](https://einvoice1.gst.gov.in/Others/APIList)** — e-Invoice generation API (mandatory for turnover > ₹5Cr).
- **[Open Government Data Platform](https://data.gov.in/)** — Public datasets useful for market research tools.
- **[DGFT APIs](https://www.dgft.gov.in/)** — Import/Export Code, RCMC, trade policy lookup.
- **[MCA21 APIs](https://www.mca.gov.in/content/mca/global/en/mca/e-filing.html)** — Company registration, director lookup, ROC filings.
- **[PM Internship Scheme](https://pminternship.mca.gov.in/)** — Government internship portal API for MSME HR automation.
- **[MSME Udyam Portal](https://udyamregistration.gov.in/)** — Udyam registration lookup and verification.

## AI & Automation

- **[n8n-io/n8n](https://github.com/n8n-io/n8n)** — Self-hostable workflow automation. Connects WhatsApp + CRM + Gmail + Sheets.
- **[activepieces/activepieces](https://github.com/activepieces/activepieces)** — Open source Zapier alternative. Good for Indian SME tool integrations.
- **[langchain-ai/langchain](https://github.com/langchain-ai/langchain)** — LLM application framework. Used for building AI assistants on top of SME data.
- **[ollama/ollama](https://github.com/ollama/ollama)** — Run local LLMs. Good for offline AI features in SME tools (no cloud cost).
- **[botpress/botpress](https://github.com/botpress/botpress)** — Open source conversational AI platform. Deploy customer-facing bots for SMEs.
- **[rasa/rasa](https://github.com/RasaHQ/rasa)** — Open source NLU + dialogue management for building Hindi/English SME chatbots.

## Developer Infrastructure

- **[supabase/supabase](https://github.com/supabase/supabase)** — Best backend stack for Indian SME SaaS builders. Postgres + Auth + Storage.
- **[pocketbase/pocketbase](https://github.com/pocketbase/pocketbase)** — Single-file backend. Good for small offline-first SME apps.
- **[nocodb/nocodb](https://github.com/nocodb/nocodb)** — Airtable alternative. Often used for SME internal tools without code.
- **[appwrite/appwrite](https://github.com/appwrite/appwrite)** — Self-hosted Firebase alternative. Auth, DB, storage for SME apps.
- **[directus/directus](https://github.com/directus/directus)** — Headless CMS + instant REST API on top of any SQL database.

## HNI & Wealth Management Tools

> Tools for high-net-worth individuals and their portfolio businesses.

- **[ghostfolio/ghostfolio](https://github.com/ghostfolio/ghostfolio)** — Self-hosted wealth management: stocks, MF, ETF, crypto. INR + NSE/BSE support.
- **[firefly-iii/firefly-iii](https://github.com/firefly-iii/firefly-iii)** — Personal finance manager for HNIs with multiple income streams and businesses.
- **[portfolio-performance](https://github.com/portfolio-performance/portfolio)** — Java-based portfolio tracker. Used by HNIs for equity + MF performance tracking.
- **[Maybe Finance](https://github.com/maybe-finance/maybe)** — Open source personal finance app with net worth tracking, good for HNI use.
- **[Actual Budget](https://github.com/actualbudget/actual)** — Self-hosted, privacy-first budgeting for HNI families and trusts.
